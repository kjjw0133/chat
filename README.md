Chat

> Delphi (Object Pascal) 기반의 채팅 프로그램

## 프로젝트 소개

Delphi의 ServerSocket,ClientSocket 컴포넌트를 활용하여 구현한 실시간 채팅 프로그램입니다.

### 개발 목적
- FireDAC을 이용한 데이터베이스 연동 구현
- Socket을 사용한 실시간 데이터 통신
- SMTP를 활용한 비밀번호 이메일 인증 구현

---

### chat 
1. 로그인 중복 방지 처리
2. 비밀번호 SHA + salt 사용하여 보안 강화
3. 비밀번호 찾기(이메일 인증) 구현
4. 소켓 통신
5. 방 생성(일반 채팅, 공개 채팅)
6. 친구인 유저를 선택하여 방을 생성, 유저를 선택할 떄 검색 창을 사용하여 유저를 빠르게 찾을 수 있음
7. 메인에서 속해있는 채팅방 바로 입장
8. 채팅 창 위치와 색을 다르게하여 상대방과 나의 채팅을 구분하기 편하게 함
9. 채팅방 처음 입장 시 메시지를 입장한 본인을 제외한 채팅방에 있는 모두에게 전송, 퇴장도 같은 동작 (12/01 추가)
10. 채팅방 정보를 볼 수 있는 창에서 채팅방에 있는 유저의 이름, 채팅방 정보를 볼 수 있음
11. 유저의 로그인을 델파이 안에서 프로그램으로 1번 처리하고, sql에서 로그인 상태를 나타내는 컬럼을 1개 만들어서 총 2개의 로그인 인증을 거쳐서 채팅방에 입장하도록 했다.
12. 메인에서 사이드 메뉴에 친구창을 만들어 친구인 유저를 확인할 수 있도록 페이지 구현
